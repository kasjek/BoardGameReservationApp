# User Stories

## User

1. As a user, I want to create new tables so that i can host board game events with the games i choose, at the time and location i want and selected the language (english, german or other (to be specified in a text field))
2. As a user, I want see all other tables offered by other users so that i can reserve seat at other tables when i do not feel like organizing one myself
3. As a user, I want to see what locations are available for booking board gaming events so that i can assess before the event if they are likely to meet my expectations
4. As a user, that creates a new table, i want to be able to define the duration of the game, event starting time, the minimum and maximum player count and i want to have a seat reserved for myself by default. 
5. As a user, I want to be able to write short reviews of other players i have played with and locations i used for my events so that i can share my opinion and help other users make friends and select the right spaces for their visits
6. As a user, I want to look up game description and picture when i click on it so that me and all other players have the same understanding of which game will be played at the event
7. As a user, I want to add pictures of the event after the event has ended so that others can see what experience i had and keep my memories of the event
8. As a user, I want to be able to chat with other players at the event chat so that in case i am running late or have other spontanious ad-hoc topics to discuss i can write to them ahead of time without exchanging our phone numbers
9. As a user, I want to be able to login with my account easy and fast so that i can see my personalized history of events i have attended and organized
10. As a user, I want to be able to update my user avatar with another random allocated one so that i find something that fits my taste
11. As a user, I want to be able to report issues to the admin in case i find something in the chat of the app obusive, rasist, nacionalistic, or in other way exclusive or harassing me so that i feel safe when using the app and inpolite users can be removed/blocked by the admin
12. As a user, I want to be able to block other players i do not wish to be in contact with so that they cannot join my tables and write to be private messages or write in the events chats to me
13. As a user, I want to be able to able to filter the events by date, time, those events that happened in the past or will happen in the future, board game name, min and max player count and location
14. As a user, I want to be able to search for other users add them as my application friends so that i can easily find them again for another event 
15. As a user, I want to be able to easily share the link of the table with my friends outside of the application so that they also register to the app and join the events and tables
16. As a user, I want to be able to invite people inside the application to join my table 
17. As a user, I want to be able to block the ability for other people to invite me to the tables so that i can freely choose if i want to use this functionality in the app or not
18. As a user, I want to be able to report a bug at any part of the application so that i can take a screenshot and write a short message to the admin to get it fixed
19. As a user, I want to be able to provide feedback at any part of the application so that admin can consider it for adding to the backlog
20. As a user, I want to be able to see the the reviews and ratings of other users when i access their profile to have a better idea if i want to become friends with them or not
21. As a user, I want to be able to cancel my seat reservation at the table up to 24h before the event takes place so that a notification is sent automatically to all other users signed up to that table 
22. As a user, who organized a table, I want to be able to cancel the table so that all people who signed up to it recevie a timely notification to their email address that the event will not take place after all
23. As a user, I want to be able to see how many events each other user have cancelled in the past so that i know how reliable to reservation with the other people is likely to be
24. As a user, who organized a table, I want to receive a confirmation from the venue that my requested table is reserved for me so that i am sure i can come on the selected day and at the selected time
25. As a user, who organized a table, I want to receive a notification and have it sent to all other players at my table in case a reservation of my table is cancelled by a venue admin so that everyone receives an early notice about it 
26. As a user, I want to be able to see other users rating, profile avatar and profile name but no other info so that privacy rules are maintained
27. As a user, I want to be able to send accept and reject friends requests from other users of the app so that i can manage who i have on my friends list
28. As a user, I want to receive notifications only when tables relevant for me (that i organize or that i reserve a seat at) make a change in time of the event, day, venue or are cancelled or missing players or have reached sufficient number of players to confirm the event 
29. As a user, I want to be able to to see what games are available at the selected venue so that i can decide if i bring my own games for a table i plan or i want to use a game available at the venue
30. As a user, I want to be able to pay for my table reservation quick and easily with paypal or revolut
31. As a user, I want to receive my money back in the the table or seat i reserved get cancelled 
32. As a user, I want to receive a notification of the successfull or insuccessfull payment
33. As a user, I want to see the status of the table as confirmed / or waiting for players / or waiting for venue confirmation so that i know what is the next step in the process at any time

## Venue Admins

34. As a venue admin, I want to be able to define the days, time and number of tables available in my own location so that i can manage the capacity of my venue at any given time
35. As a venue admin, I want to be able to accept or reject the reservation requests sent by the users so that all of us have visibility whether the event can take place (and table can be reserved with or without a requested game)
36. As a venue admin, I want to be able to modify the description of my location and pictures, so that i can manage the online image of my business in the application
37. As a venue admin, I want to respond to the reviews posted at my location by the users so that i can get more information about their actual experience and reasons of behind their review in case they are missing from the description itself
38. As a venue admin, I want to define and edit the games available in my place so that people know what is available for renting when visiting
39. As a venue admin, I want to write reviews of the players that have visited my location so that i keep track of the guests that were already visiting and how they behaved 
40. As a venue admin, I want to be able to see all the tables reserved in the past at my location to be able to use that data for any advertisment i plan for my own business
41. As a venue admin, I want to be able to see all the events planned to happen at my location in a single space to be able to manage them from a single page, screen
42. As a venue admin, I want to be able to block users from booking at my venue again in case i find the users behaviour inappropriate or obusive with providing a reason in the blocking message for that and notify the admin
43. As a venue admin, I want to receive notifications in case of new reservation reqests, edited requets and cancelled requests for my venue so that i can adjust my capacity plan
44. As a venue admin, I want to be able to manage the games inventory in my location and update the number of copies of the games available, their state (new/like new/in good shape) and language they are available in
45. As a venue admin, I want to be able to add, edit and remove reservation rules for my location so that people know what is the maximum reservatin time and if people are requested to buy food or drinks only are OK and if there is a minimum spent requetsed per person or not.

## Admin

46. As admin, I want to be able to do everything that users and venue admin do but also to add, edit, remove locations and venue admins so that i can adjust my business as i need
47. As admin, I want to be able to add, edit and remove games from the inventory of any venue so that the overview remains up to date
48. As admin, I want to be able to delete and obusive comments, responses, posts, pictures so that all my users feel safe
49. As admin, I want to receive notifications of obusive/rasist/nationalistic behaviors so that i can react and block respective users or locations
50. As admin, I want to receive reservation fees from the tables my users book, so that i can finance my business operations
51. As agmin, I want to have overview of the fees on a single space per location, per user and per game so that i know what is worth investing more into and what not
52. As admin, I want to see be notified of any bad review for a user or location so that i can timely verify it and react to it if needed 
53. As admin, I want to be able to block any user or location that do not follow the rules of using the app so that everyone feels safe while using it
54. As admin, I want to be able to mark locations or users as "super locations" or "super users" if i notice that reviews of those are very very good
